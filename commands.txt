git pull //отримати оновлення з гітхаб на локальний комп'ютер
git push //надіслати коміт на гітхаб
git push -u origin master //встановити репозиторій і гілку і надіслати на гітхаб
git remote add origin https://github.com/serhiypikulya/udemy.git
git commit -a -m "second commit" //створити контрольну точку
git add --all //відслідковувати всі файли проекту 
git status //перегляд статусу файлів в проекті
git add *.css //відслідковувати обрані файли проекту 
git init //ініціювати гіт
В первой строке введите комментарий: «..». Сохраните файл и выйдите из редактора (для этого в редакторе по умолчанию (Vim) вам нужно нажать клавишу ESC, ввести :wq и нажать Enter)
git checkout commit //Відновити коміт 
    Где commit - это хеш (обозначение, имя) коммита, причем можно указывать не весь хеш, а несколько начальных символов хеша.
